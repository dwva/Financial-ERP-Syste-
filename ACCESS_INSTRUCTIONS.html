Financial ERP System Access

Local Network Access

Access the application from any device on the same network using these links:

Main Application: http://10.143.139.79:8090 File Upload Server: http://10.143.139.79:3002

Access Instructions

  1. Ensure both servers are running:
    • Main application server (port 8090)
    • File upload server (port 3002)
  2. Connect from another device:
    • Make sure the other device is on the same WiFi/network
    • Open a web browser on that device
    • Navigate to http://10.143.139.79:8090

⚠️ Security Warning

These servers are accessible to anyone on your local network. For security:

  • Don't share your network password with untrusted users
  • Stop the servers when not in use
  • Consider using strong passwords for admin accounts

Troubleshooting

If you can't access the application from another device:

  • Check that both servers are still running
  • Verify the IP address with ipconfig (Windows) or ifconfig (Mac/Linux)
  • Check Windows Firewall settings to ensure ports 8090 and 3002 are allowed
  • Try temporarily disabling antivirus software that might block the connection
